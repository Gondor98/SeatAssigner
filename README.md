# 💒 Wedding Seat Assigner

Ứng dụng web xếp chỗ ngồi tiệc cưới - Khách mời quét QR để tìm bàn của mình.

## Tính năng
- 📋 Upload danh sách khách mời từ file Excel
- 🪑 Cấu hình bố trí bàn tiệc (hàng/cột/số lượng)
- 📱 Tạo mã QR cho khách quét
- 🔍 Tìm kiếm tên thông minh (hỗ trợ dấu tiếng Việt)
- 🗺️ Hiển thị bản đồ 2D vị trí bàn

## Cách sử dụng
1. Mở trang Admin → Upload file Excel → Cấu hình bàn → Đẩy lên Cloud
2. Tạo mã QR → In ra đặt ở lối vào
3. Khách quét QR → Nhập tên → Thấy số bàn + bản đồ

## Tech Stack
- Pure HTML/CSS/JavaScript (no framework)
- Google Sheets API (database)
- GitHub Pages (hosting)
